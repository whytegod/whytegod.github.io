Protocol Registry

This registry documents verifiable on-chain state changes across major blockchain protocols.

Each record reflects a confirmed, irreversible modification to protocol state, permissions, or execution logic.

WHYTEGOD does not interpret, rank, predict, or speculate. Records exist solely to preserve state history.

Registry Mechanism

Protocols are included in this registry only after a verifiable on-chain event produces a permanent change to system behavior.

  • No off-chain announcements
  • No roadmap promises
  • No narrative-driven updates

If a state change cannot be cryptographically verified, it does not exist here.

State Change Classification

  • Tier 1: Core execution logic or governance authority change
  • Tier 2: Parameter, configuration, or risk model change
  • Tier 3: Peripheral or auxiliary system deployment

Only Tier 1 and Tier 2 events are currently recorded.
