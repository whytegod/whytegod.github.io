Frequently Asked Questions

This page answers common questions about the Whytegod Protocol and WGD token usage in simple terms.

What is Whytegod?

Whytegod is a neutral protocol for creating and verifying permanent records using cryptographic hashes and public anchors.

Does Whytegod store my data?

No. The protocol never stores user data. Only cryptographic hashes are used for verification.

What is WGD?

WGD is a utility token used to introduce cost and prevent abuse when anchoring records in the protocol.

What is a WGD Token Member?

A WGD Token Member is any person or entity that holds WGD for the purpose of using the Whytegod Protocol.

Does holding WGD give me control or voting rights?

No. Holding WGD does not grant governance, voting, ownership, or decision-making authority.

Is WGD an investment?

No. WGD is not an investment contract, security, or profit-sharing instrument. It exists only for protocol utility.

Do I need WGD to verify records?

No. Verification is free and permissionless. WGD is only required for anchoring new records.

Can the protocol rules change?

Whytegod v1.0 is frozen. Any future changes require a new version.

Whytegod defines technical rules only. Interpretation and usage are the responsibility of users.
