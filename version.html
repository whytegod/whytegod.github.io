Protocol Version

This page defines the immutable version of the Whytegod Protocol currently in effect.

Current Version

Whytegod Protocol v1.0

Release Status

  • Status: Final
  • Governance: None
  • Mutability: Frozen

What v1.0 Defines

  • WRID generation using SHA-256
  • Hash-only record representation
  • Public immutable anchoring
  • Permissionless verification
  • WGD utility-based cost model
  • No governance or control layer

What v1.0 Does NOT Include

  • No consensus mechanism
  • No on-chain storage
  • No identity system
  • No dispute resolution
  • No upgrade path

Immutability Policy

Version 1.0 is permanently frozen. Its rules, semantics, and guarantees cannot be modified.

Future Versions

Any future protocol changes must be released as a separate, explicitly labeled version.

Adoption of new versions is voluntary. No migration is required or enforced.

Backward Compatibility

Records anchored under v1.0 remain valid forever, regardless of future versions.

This version declaration is final. The protocol is complete.
