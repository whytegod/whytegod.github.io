Protocol Version

This page defines the official versioning of the Whytegod Protocol.

Current Version

Whytegod Protocol v1.0

  • Status: Active
  • Release Type: Initial stable specification
  • Governance Model: Frozen

Version Philosophy

Whytegod follows strict version isolation. A version defines immutable rules.

  • Versions do not change after release
  • Records remain bound to their version
  • Verification depends on version context

Backward Compatibility

New versions do not modify or invalidate previous records.

Older versions remain verifiable forever.

Future Versions

Future versions may introduce:

  • Alternative anchoring methods
  • Extended record formats
  • Optional tooling improvements

Adoption of future versions is voluntary.

Forking

Anyone may implement or fork the protocol. No permission is required.

Market adoption determines relevance.

Canonical Reference

This website serves as a public reference for the Whytegod Protocol specification.

The protocol does not depend on the website for operation or verification.

Legal Scope

Version definitions describe technical behavior only. They do not create legal obligations or guarantees.

Versioning preserves certainty. Stability is a feature, not a limitation.
